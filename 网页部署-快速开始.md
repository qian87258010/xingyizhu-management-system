# 🌐 3分钟部署到网页

## 🚀 最简单的方式：Vercel（推荐）

### 步骤1️⃣：上传代码到 GitHub

```bash
# 如果还没有 git 仓库
git init
git add .
git commit -m "首次提交"

# 在 GitHub 创建新仓库后
git remote add origin https://github.com/你的用户名/星奕筑管理系统.git
git push -u origin main
```

### 步骤2️⃣：部署到 Vercel

1. 访问 **[vercel.com](https://vercel.com)**
2. 点击 **"Sign Up"** 用 GitHub 登录
3. 点击 **"New Project"**
4. 选择你的仓库 **"星奕筑管理系统"**
5. 点击 **"Deploy"**
6. ☕ 等待 1-2 分钟...
7. ✅ 完成！你会得到一个网址

## ❓ 常见问题

### Q: 部署后数据会丢失吗？
**A:** 不会！数据存储在浏览器的 LocalStorage 中，每个用户有自己的数据。

### Q: 可以用自己的域名吗？
**A:** 可以！在 Vercel/Netlify 控制台配置自定义域名。

### Q: 需要付费吗？
**A:** 不需要！Vercel、Netlify、GitHub Pages 都提供免费额度，足够使用。

### Q: 如何更新网站？
**A:** 只需 `git push`，会自动重新部署！
