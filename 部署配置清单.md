## 🎨 自定义配置

### 修改项目名称

**Vercel/Netlify：**
1. 进入项目设置
2. 修改 Site Name
3. 网址更新为：`https://new-name.vercel.app`

### 绑定自定义域名

**步骤：**
1. 在平台添加域名
2. 配置 DNS 记录
3. 等待生效（5-30分钟）

**DNS 配置示例：**
```
类型: CNAME
名称: www
值: cname.vercel-dns.com
```

## 🔐 安全配置

### 已包含的安全头

**vercel.json / netlify.toml 已配置：**

```
✅ X-Content-Type-Options: nosniff
✅ X-Frame-Options: DENY

```

### 建议

- ✅ 使用 HTTPS（自动启用）
- ✅ 定期更新密码
- ✅ 不在公网存储敏感数据
- ⚠️ 如需真实业务使用，建议连接数据库
