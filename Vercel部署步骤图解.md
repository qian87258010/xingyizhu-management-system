## 🎨 自定义配置（可选）

### 1. 修改项目名称

在 Vercel 控制台：
1. 进入项目设置（Settings）
2. 找到 "Project Name"
3. 输入新名称
4. 保存

**新网址：**
```
https://新名称.vercel.app
```

### 2. 绑定自定义域名

如果您有自己的域名（如 `xingyizhu.com`）：

1. 在 Vercel 项目中点击 **"Domains"**
2. 点击 **"Add Domain"**
3. 输入您的域名：`www.xingyizhu.com`
4. 按照提示添加 DNS 记录：
   ```
   类型: CNAME
   名称: www
   值: cname.vercel-dns.com
   ```
5. 等待 DNS 生效（5-30分钟）
6. 完成！

## ❓ 常见问题解决

### Q1: 部署失败，显示 "Build Failed"

**解决方案：**
1. 点击查看构建日志
2. 检查错误信息
3. 常见原因：
   - 依赖安装失败 → 检查 `package.json`
   - 构建命令错误 → 确认是 `npm run build`
   - 代码语法错误 → 在本地先测试 `npm run build`

### Q2: 网站打开是空白页

**解决方案：**
1. 按 F12 打开浏览器控制台
2. 查看错误信息
3. 检查：
   - 路由配置是否正确（已在 `vercel.json` 中配置）
   - 静态资源是否加载成功

### Q3: 看不到自己的 GitHub 仓库

**解决方案：**
1. 在导入页面点击 **"Adjust GitHub App Permissions"**
2. 授权 Vercel 访问特定仓库或所有仓库
3. 刷新页面

### Q4: 登录后刷新页面会退出

**说明：** 这是正常现象！
- 当前版本使用前端模拟数据
- 数据存储在浏览器的内存中
- 刷新页面后需要重新登录
- 如需持久化登录，建议连接 Supabase 数据库

### Q5: 数据在不同浏览器/设备不同步

**说明：** 这是正常现象！
- 数据存储在浏览器本地
- 不同设备/浏览器的数据是独立的
- 如需多设备同步，建议连接 Supabase 数据库
