## 🚀 快速开始

### 最简单的方式（推荐）

在项目根目录打开终端，运行：

```bash
chmod +x quick-start.sh

```

然后选择对应的选项即可！

## 📋 详细步骤

### 第一步：安装基础依赖

```bash
# 确保你已经在项目目录中
npm install
```

### 第二步：安装 Electron 打包工具

```bash
npm install --save-dev electron electron-builder concurrently wait-on cross-env
```

或者运行快捷脚本：
```bash
chmod +x scripts/install-electron.sh
./scripts/install-electron.sh
```

### 第三步：打包应用

**方法 A：使用 npm 命令**
```bash
npm run dist:mac
```

**方法 B：使用打包脚本**
```bash
chmod +x scripts/build-mac.sh
./scripts/build-mac.sh
```

**方法 C：使用快捷启动工具**
```bash
./quick-start.sh
# 选择选项 4
```

### 打包时间
- 首次打包：5-10 分钟（需要下载 Electron）
- 后续打包：2-3 分钟

### 打包产物

打包完成后，在 `release/` 文件夹中会生成：

```
release/
├── 星奕筑管理系统-0.0.1-arm64.dmg      ← Apple Silicon Mac 安装包
├── 星奕筑管理系统-0.0.1-x64.dmg        ← Intel Mac 安装包
├── 星奕筑管理系统-0.0.1-arm64-mac.zip
└── 星奕筑管理系统-0.0.1-x64-mac.zip
```

## 💻 安装使用

### 在你的 Mac 上安装

1. **找到对应的 DMG 文件**
   - M1/M2/M3 Mac → 使用 `arm64.dmg`
   - Intel Mac → 使用 `x64.dmg`

2. **双击打开 DMG 文件**

3. **拖动到 Applications**
   - 将应用图标拖到"应用程序"文件夹

4. **首次运行**
   ```
   ⚠️ 重要：首次运行需要特殊操作
   ```
   - 打开"应用程序"文件夹
   - 找到"星奕筑管理系统"
   - **右键点击** → 选择"打开"
   - 在弹出对话框中点击"打开"
   
   只需首次这样做，之后可以直接双击打开。

### 绕过安全检查（可选）

如果右键打开也不行，在终端运行：

```bash
sudo xattr -r -d com.apple.quarantine /Applications/星奕筑管理系统.app
```

输入你的 Mac 密码确认。

## ❓ 常见问题

### Q1: 打包后文件太大了？
**A:** 这是正常的。Electron 应用包含：
- 你的应用代码（几 MB）
- Chromium 浏览器（约 100MB）
- Node.js 运行时（约 20MB）

完整应用约 150-200MB 是正常的。

### Q2: 可以打包 Windows 版本吗？
**A:** 可以，但需要在 Windows 电脑上打包，或使用 CI/CD 服务。
修改 `package.json` 中的 `build` 配置添加 Windows 支持。

### Q3: 应用数据存储在哪里？
**A:** 
```
~/Library/Application Support/星奕筑管理系统/
```
LocalStorage 数据会自动保存在这里。

### Q4: 如何更新应用版本？
**A:** 
1. 修改 `package.json` 中的 `version` 字段
2. 重新打包：`npm run dist:mac`
3. 新版本会有新的文件名

### Q5: 卸载应用
**A:** 
1. 打开"应用程序"文件夹
2. 找到"星奕筑管理系统"
3. 拖到废纸篓
4. 清空废纸篓

如果想彻底清理数据：
```bash
rm -rf ~/Library/Application\ Support/星奕筑管理系统
```

### Q6: 打包失败怎么办？
**A:** 尝试以下步骤：
```bash
# 1. 清理缓存
rm -rf node_modules dist release

# 2. 重新安装依赖
npm install

# 3. 安装 Electron 依赖
npm install --save-dev electron electron-builder

# 4. 重新打包
npm run dist:mac
```

### Q7: 可以在 M1 Mac 上打包 Intel 版本吗？
**A:** 可以！默认配置会同时打包两个架构的版本。
